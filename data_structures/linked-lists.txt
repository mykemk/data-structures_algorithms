 - It is a collection of data elements called nodes
 - Each node has a field that points to the next element in a list
 - The last item in the list has a field that points to nothing
 - A singly linked list has only one direction of items provided

 ** Pros **
 - It is fast and easy to add and remove items from the list
 - Underlying elements doesn't have to be re-organised

 ** Cons **
 - Can't do constant time random item access
 - Item lookup is linear in time complexity
